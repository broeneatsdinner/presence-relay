# Sanitized Operator Transcripts

These presentation fixtures show the verified human-readable shape of the
private read-only operator interface. They are synthetic documentation examples,
not captured live output and not generated by `demo/bin/demo.sh`.

- [`doorway-arrivals.example.txt`](doorway-arrivals.example.txt) shows one
  matched and one unmatched home-arrival window.
- [`doorway-departures.example.txt`](doorway-departures.example.txt) shows one
  matched and one unmatched doorway-button window.

The Unicode box tables, em dashes, timestamps, and IDs are safe public fixture
material. JSON mode remains machine-readable and represents unmatched fields as
`null`; these files intentionally focus on the human operator experience.
